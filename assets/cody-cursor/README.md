# Cody cursor assets

SVG pointers for the fullscreen overlay demo (follow mouse → point at source).

| File | Use | Hotspot (viewBox px) |
|------|-----|----------------------|
| `cursor-default.svg` | Follow mode | `(6, 4)` tip |
| `cursor-point.svg` | Point / arrive mode | `(6, 4)` tip |

**Brand:** fill `#1B4FD8`, outline `#0A1628`, face wash `#E8F0FE`.

**Demo:** open [`../../demo/cody-cursor/host.html`](../../demo/cody-cursor/host.html) (iframe spans the viewport). Overlay page alone: [`../../demo/cody-cursor/overlay.html`](../../demo/cody-cursor/overlay.html).
