# Cody cursor assets

Required for the Clicky-style desktop cursor (scope design 2026-07-18).

| File | Purpose |
|------|---------|
| `cursor-default.png` | Resting / follow sprite |
| `cursor-point.png` | Optional pointing frame |

**Hotspot:** tip of the pointer (document pixel coords when art lands).

Until final art is dropped here, `overlay.point_at` uses a geometric OS pointer animation fallback.
