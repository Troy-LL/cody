Cody

It points to the file.

Cody is an AI that shows you where something is instead of telling you how to find it.

View on GitHub

How it works

  1. 1

    Ask

    Describe a file in natural language — Taglish, English, messy phrasing welcome.

  2. 2

    Match

    Cody reasons over your files and picks the one that fits, with a clear why.

  3. 3

    Point

    Your OS reveals the file and Cody speaks the confirmation — no folder-by-folder instructions.

The demo moment

Watch Explorer select the file while Cody says the name aloud. No guessing which window, no scrolling through Downloads — just a finger on the file.

See the code

Cody is open source. Clone the repo, run the demo, and point at files on your own machine.
